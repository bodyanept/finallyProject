# Необходимые элементы сайта

### Удовлетворительно:
  - БД
  - 1 api производителя
  - home/каталог/страница товара
### Хорошо:
  - 2 api производителя
  - поиск
  - корзина
  - оплата
  - фильтр по VIN
  - панель админа (редактирование цены товара)
  - контроль версий git
### Отлично:
  - 5 api производителя
  - доставка (код отслеживания)
  - фильтр конкретизации
  - профиль покупателя
  - гараж

### Дополнительно:
  - Api реальных служб доставки
  - планировщик цен
  - прочее
